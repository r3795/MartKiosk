## 💡 Development Features
- 마트에서 물건을 구매할 때 도움을 주는 키오스크 프로그램
  
- 모든 함수는 리턴 값이 없고 함수 자체적으로 해결한다.

- 입력 값이 정수가 아닌 경우 예외처리를 한다.

- while문, for문을 이용해 원치 않는 값이나 행동 진행 시 알맞은 값이 입력될 때까지 반복한다.

- format함수를 이용하여 마지막 최종 연산 결과를 소수점 제외, 천 단위까지 표시한다.

- dict 딕셔너리를 이용해 key와 value에 값을 저장하여 동시에 사용한다.

- 로그인 기능을 이용해 로그인 시 기능 이용 또는 회원가입 후 기능 이용 절차를 진행한다.

- 고객이 물품 위치를 궁금해 할 경우 Turtle함수를 이용하여 경로를 안내한다.

- 대여 물품과 구매 물품을 따로 관리하기 때문에 물품 위치 검색 시에도 구역을 매개변수로 받아 조건에 맞는 함수를 실행한다.

- 딕셔너리에 key과 value값을 새로 삽입하여 장바구니를 관리한다.

- datetime 모듈을 이용하여 물품을 구매한 시각을 정확히 기록한다.

- 리스트를 이용하여 추가된 물품 별 개수의 값도 별도 저장한다.

- 고객이 여러 상품을 여러 개수로 구매할 수 있기 때문에 상품에 따른 개수를 따로 저장하고 이후 결제 시 계산하여 총 금액을 출력한다.

- 물품 선택 후 미 구매 시 프로그램을 종료한다.

- 최초 로그인 진행 후 물품이 출력되고 INPUT문에 따라 구매 또는 탐색을 진행한다.

- 물품 구매 시 물품명과 개수를 입력받아 저장하고 계속 구매할 것인지 묻는다. 이때 계속 구매를 한다면 구매하는 물품과 개수를 모두 저장한 후 마지막 결제 시 계산하여 산출한다. 이 결과는 영수증 출력 시에도 활용된다.

- 최종적으로 여태껏 담은 물품에 대한 수량과 가격을 확인하고 소계로 총금액을 출력한다.

- 이 때, 형태는 영수증 형태로 출력한다.

- 물품 탐색 시 검색하고 싶은 물품명을 입력받아 해당 물품의 위치를 거북이를 활용하여 표시한다. 해당 위치는 빨간색 사각형을 그리며 강조 표시된다.

- 딕셔너리에 key와 value를 동시에 삽입하는 것을 구현했다.
